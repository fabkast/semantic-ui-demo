Características principales

Diseño adaptable

Diseñado completamente con em y rem.

Auto explicativo

Uso de lenguaje simple y común.

Indiferente a etiquetas

Se puede usar cualquier etiqueta con los elementos UI.

Fuerte enfoque al uso de patrones y la reusabilidad

Las definiciones de elementos pueden ser expresadas en grupos y tienen atributos compartidos evitando declaraciones repetitivas.

Portable e independiente

Usa sólo los componentes que necesites.

Javascript

Incluye componentes javascript, librería de animaciones y validación de formularios.

Preprocesadores CSS

Usa LESS y tiene un port oficial a SASS.

Documentación extensiva

Encuéntrala en su sitio: semantic-ui.com

Definitions

Las definiciones (definitions) en Semantic proveen de un formato reusable para compartir un diseño de interfaz.

Se tratan de grupos de css, fuentes, imágenes y javascript que conforman un sólo elemento.

Tipos de definiciones

UI Elements

Bloques básicos. Existen por sí solos o en grupos homogéneos.

Button Divider Header Icon Image Input Label Loader Progress Reveal Segment Step

UI Collections

Grupo heterogéneo de varios componentes que usualmente pueden ser encontrados juntos.

Breadcrumb Form Grid Menu Message Table

UI Views

Una manera de presentar contenido común.

Comment Feed Item List

UI Modules

Elementos que se definen esencialmente por su comportamiento.

Accordion Checkbox Dimmer Dropdown Modal Popup Rating Shape Sidebar Transition

UI Behaviors

Elementos que se definen esencialmente por su comportamiento.

Form validation

Código

<!-- HTML -->
<form class="ui form segment" style="display: block;">
  <p>Let's go ahead and get you signed up.</p>
  <div class="two fields">
    <div class="field">
      <label>First Name</label>
      <input placeholder="First Name"
          name="first-name" type="text">
    </div>
    <div class="field">
      <label>Last Name</label>
      <input placeholder="Last Name"
          name="last-name" type="text">
    </div>
  </div>
  <div class="field">
    <label>Username</label>
    <input placeholder="Username"
        name="username" type="text">
  </div>
  <div class="field">
    <label>Password</label>
    <input type="password" name="password">
  </div>
  <div class="inline field">
    <div class="ui checkbox">
      <input type="checkbox" name="terms">
      <label>I agree to the terms and conditions</label>
    </div>
  </div>
  <button class="ui blue submit button" type="submit" >Submit</button>
</div>
</div>
</form>

<!-- Javascript -->
<script>
$('.ui.form')
.form({
  firstName: {
    identifier  : 'first-name',
    rules: [
      {
        type   : 'empty',
        prompt : 'Please enter your first name'
      }
    ]
  },
  lastName: {
    identifier  : 'last-name',
    rules: [
      {
        type   : 'empty',
        prompt : 'Please enter your last name'
      }
    ]
  },
  username: {
    identifier : 'username',
    rules: [
      {
        type   : 'empty',
        prompt : 'Please enter a username'
      }
    ]
  },
  password: {
    identifier : 'password',
    rules: [
      {
        type   : 'empty',
        prompt : 'Please enter a password'
      },
      {
        type   : 'length[6]',
        prompt :
            'Your password must be at least 6 characters'
      }
    ]
  },
  terms: {
    identifier : 'terms',
    rules: [
      {
        type   : 'checked',
        prompt :
          'You must agree to the terms and conditions'
      }
    ]
  }
  },
  { inline: true }
);
</script>
